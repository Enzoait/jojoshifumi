# Jojoshifumi ⛰️📄✂️
Un jeu de Shifumi fait en React.

# Installation 

Lancer le serveur :

```shell
cd chifoumi-server-main
cd functions
docker compose up
```
Lancer l'environnement de dev :
```shell
cd ..
cd ..
cd jojoshifumi
npm install
npm run dev
```
